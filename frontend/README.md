# Upsum Frontend

This directory contains the React + Vite frontend for Upsum.

## Setup

1. Install dependencies:
   npm install

2. Start development server:
   npm run dev

3. Open http://localhost:5173 to view the app.

## Structure
- Main UI for search and results
- Connects to backend API (http://localhost:8000)
- Nordic-inspired, dark-first design
